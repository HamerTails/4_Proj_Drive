# API – SUPFILE

L’API constitue le coeur de la plateforme SUPFILE.
Elle gère l’authentification, la gestion des fichiers et des dossiers,
ainsi que les fonctionnalités de partage.

Fonctionnalités prévues :
- Inscription et connexion des utilisateurs
- Authentification sécurisée via JWT et OAuth2
- Gestion des dossiers et fichiers
- Upload et téléchargement de fichiers
- Prévisualisation des fichiers (images, PDF, textes)
- Partage public et interne

Technologies :
- Node.js
- Express
- PostgreSQL
- Stockage des fichiers sur disque (volume Docker)
